# 2D Graphics Basics


## Shapes
* Rectangles
* Ellipses
* Lines
* Polygons
* Paths - the general case, nodes and curves


## Properties
Shapes have many properties including:

* fill
* stroke 
* stroke-style
* width
* color
* location


## Transformations
apply to single or multiple shapes

* scaled
* rotated
* translated
* skewed


## Group Operations
apply to multiple shapes

* group/ungroup
* align
